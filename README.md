# wyuIoT
基于esp8266单片机，适用于五邑大学智能宿舍，自动开关灯自动开门

## 补充说明
1.开发板5v供电，属于小功率电器，符合宿舍管理条例  
2.宿舍自动开灯的物料与开门一致（不用买铁线/铜线）  
3.电源请使用usb插线板，自备热熔胶枪以固定舵机
